Key Decisions

  • Single process: npx chat-browser starts Express, which serves the pre-built React SPA and the REST API on one port
  • SQLite via better-sqlite3: synchronous, fast, no native build issues on most platforms. FTS5 for full-text search.
  • Chokidar: watches session directories for new/modified files, updates the index incrementally
  • Pre-built SPA: React + Vite built at npm publish time, shipped as static assets — no build step on install
  • CLI flags: --port, --open (auto-open browser), --reindex (rebuild from scratch), --data-dir (override session locations)
